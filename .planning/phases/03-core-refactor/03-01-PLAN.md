# Plan 03-01: Config Integration — page.tsx + navbar + whatsapp

## Goal
Connect page.tsx to use `sections` config for conditional rendering, navbar to use `brand`/`navLinks`/`contact.whatsapp`, and whatsapp-button to use `whatsapp` config.

## Tasks
1. Update `app/page.tsx`: import config, conditionally render sections based on `config.sections`
2. Update `navbar.tsx`: import config, read `brand.name`, `brand.slogan`, `navLinks`, `contact.whatsapp`
3. Update `whatsapp-button.tsx`: import config, read `whatsapp.number` and `whatsapp.message`

## Verification
- Changing brand.name in config updates navbar text
- Disabling a section in config hides it from the page
- WhatsApp button uses number from config
