# Plan 03-02: Config Integration — hero + services + footer + contact

## Goal
Connect hero, services, footer, and contact-form components to read data from config.

## Tasks
1. Update `hero.tsx`: read `hero.badge`, `hero.headline`, `hero.subheadline`, `hero.stats`, `hero.tags`, `hero.ctas` from config
2. Update `services.tsx`: read `services` array from config, use `getIcon()` to resolve icon strings
3. Update `footer.tsx`: read `brand`, `footerLinks`, `footerServices`, `social`, `contact` from config
4. Update `contact-form.tsx`: read `contact.phone`, `contact.email`, `contact.hours`, `services` from config

## Verification
- Changing hero headline in config updates the rendered hero
- Services section renders from config array
- Footer shows brand name, links, and social from config
- Contact form shows phone/email from config
