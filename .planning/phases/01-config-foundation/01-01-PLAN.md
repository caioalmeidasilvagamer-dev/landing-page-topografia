Create `lib/config-types.ts` with ALL 23 section schemas defined as separate named Zod object schemas, plus the top-level SiteConfigSchema that composes them. Export both the schema and the inferred TypeScript type.

**File structure (top to bottom):**
1. Import: `import { z } from 'zod'`
2. JSDoc comment explaining this file is the single source of truth for all config types
3. Individual section schemas (each as `export const XxxSchema = z.object({...})`)
4. Top-level `export const SiteConfigSchema = z.object({...})` composing all sections
5. Export: `export type SiteConfig = z.infer<typeof SiteConfigSchema>`
